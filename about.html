IT 4203 Term Project

Project Details

In this term project, we apply all the knowledge and skills developed throughout the semester and created a single page weather application. The knowledge and skills applied include jQuery, JSON, RESTful Web API, AJAX, and Bootstrap to include a rich UI design. We utilized the OpenWeather and the OpenCage API to create a website that will allow the user to see the current weather for a city and state.
